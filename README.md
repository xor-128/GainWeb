# GainWeb
Gain.TV Resmi olmayan web uygulaması.

Canlı test-sunucusu: http://gain-web.ml/

# Özellikler  
 - Basit search sistemi
 - Temiz UI
 - Çoklu kalite seçeneği
 - Playlist desteği
 - Paylaşma için arama ve playlist bazlı url sistemi.

# Yapılacaklar
